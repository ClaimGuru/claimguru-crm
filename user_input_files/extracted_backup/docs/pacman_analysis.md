# Pacman Competitor Analysis

**Website:** [https://www.pacmansoftware.com/](https://www.pacmansoftware.com/)

## Core Features
- **Claims Management:** Cloud-based claims management designed for public adjusters and attorneys.
- **Individual Login/Tracking:** Adjusters, solicitors, and customers can be granted access to see only the claims they are assigned to.
- **Claim Diary:** Time and date stamped entries for tracking claim progress.
- **Document Management:** Upload documents, letters, and photos within each claim.
- **Email Integration:** Send and receive emails directly through the platform.
- **Templates:** Customizable templates for invoices, letters, and emails.

## Pricing Structure
- **Single User:** $69/month.
- **Pro (up to 5 users):** $199/month.
- **Enterprise (up to 10 users):** $299/month.
- **Enterprise Plus (over 10 users):** Custom pricing.

## AI/Automation
- **No Mention of AI:** The website does not mention any AI or machine learning capabilities.

## Integration Capabilities
- **No Mention of Integrations:** The website does not mention any third-party integrations.

## Security Features
- **"Extremely Secure":** The website claims the software is "extremely secure" but provides no specific details.

## Data Migration
- **Free Data Import:** The company offers free data import from other software, which takes 5-7 business days.

## Mobile Features
- **Mobile App:** A free mobile app is available for iOS and Android.

## Unique Selling Points
- **Free Data Migration:** Offering free data import is a significant differentiator.
- **Mobile App:** A dedicated mobile app is a key feature.
- **Granular Access Control:** The ability to provide access to clients and solicitors with restricted views is a strong feature.

## Weaknesses/Gaps
- **No AI/Automation:** The platform lacks the AI and automation features that are becoming increasingly common in the market.
- **Lack of Integration Information:** No details on integrations with other software.
- **Vague Security Claims:** The claim of being "extremely secure" is not backed up with any specific details.
